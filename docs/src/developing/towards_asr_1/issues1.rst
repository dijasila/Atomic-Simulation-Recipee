======
Issues
======

This has worked quite well for some time, but there are some
deficiencies in the current implementation, namely, when a recipe is
to be run several times. This can occur in

  - Convergence studies
  - Stiffness tensor calculations
  - Piezo electric tensor calculations

Other issues

  - Inflexible dependency system, prone to human error
  - Inflexible file creation system, prone to human error
